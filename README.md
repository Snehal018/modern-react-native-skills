# modern-react-native-skills

React Native and Expo skills for predictable agent-driven app work.

## Current Skills

- `setup-react-native`: baseline setup for an existing Expo or React Native app.
- `add-react-native-feature`: feature, screen, flow, form, and API-backed module implementation.
- `rn-navigation-flow`: navigation routes, params, deep links, gates, and route lifecycle behavior.
- `rn-api-data`: API clients, service functions, TanStack Query hooks, mutations, cache, and external data validation.
- `rn-forms-validation`: React Native forms, schemas, field errors, submit states, and keyboard-aware input flows.
- `rn-auth-secure-storage`: auth flows, token lifecycle, protected routes, SecureStore, auth headers, and cache clearing.
- `rn-quality-review`: React Native quality reviews and targeted fixes.

## Foundation

- `skills/_shared/skill-system-constitution.md` defines the skill-system rules and contribution gate.
- `skills/_shared/react-native-modern-standard.md` defines the shared React Native app-code standard.
- `docs/wave-0-foundation.md` records ideal skill invocation examples and the Wave 0 `agents/openai.yaml` decision.
- `docs/wave-1-everyday-app-work.md` records the Wave 1 skill boundaries and completion criteria.
- `docs/react-native-skill-system-roadmap.html` shows the broader build path.

Wave 0 is complete. Wave 1 is complete with the everyday app-work skills for navigation, API/data, forms, and auth/secure storage.
